#### 群编码解读
##### 第一位
D是Dolphin首字母。

##### 第二位
该位字母代表了群字母，如A或B等字母。

##### 第三位
若为1，则该群为医务所官方建立的群聊。

若为2，则为其他人建立的群聊，后来被纳入医务所。

##### 第四位
若为1，则代表本群为配置发布专用群。

若为2，则代表本群为交流群。

若为3，则代表本群为综合性群聊。

##### 第五位
若为1，则代表本群为受到特殊保护的群聊，通常这种群聊存储了前人离开后留下的“遗物”，仅允许管理组核心成员管理。

若为2，则代表本群不是受特别保护的群聊，可由群管理员或管理组其他成员一起管理。
